# Snake

This is a very simple snake game written in Python with PyQt5, and in C++ with Qt5.

![Screenshot](./screenshot.png)

## Quickstart

### Python

```bash
python -m venv venv
. ./venv/bin/activate
python -m pip install -r requirements.txt
./main.py
```

### C++

```bash
qmake
make
./main
```
